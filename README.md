# 📷 照片歸檔工具 (Photo Archiver) v8.13

這是專為行政人員、活動紀錄整理設計的純網頁小工具。
**無需安裝任何軟體**，只要有瀏覽器就能用。它可以幫您用最快的速度檢視大量照片，只靠鍵盤就能完成分類、刪除與命名，最後自動產生整理腳本與檔名參照表。[https://progress01.github.io/photocheck/](https://progress01.github.io/photocheck/)

---

## 📌 這個工具能幫您做什麼？

如果您手邊有幾百張活動照片，需要處理以下繁瑣工作：

1. **挑選照片**：剔除模糊、閉眼、重複的廢片。
2. **分類歸檔**：依照「活動」、「會議」、「場勘」將照片分到不同資料夾。
3. **統一命名**：將無意義的 `DSC_1234.jpg` 自動改為 `活動_長官致詞_001.jpg`。
4. **微調檔名**：某些照片不想用分類命名，想直接保留原檔名或手動微調。
5. **文書報告**：需要將照片檔名複製貼上到 Word 報告中。

這個工具讓您在一個畫面中完成所有決策，並自動產出執行檔，幫您把檔案搬好、名字改好。

## 🛠️ 使用方法 (SOP)

### 第一步：開啟工具

1. 下載本專案的 `index.html` 檔案。
2. **請將此檔案放入您存放照片的資料夾中**（也就是照片的最外層）。
3. 直接點兩下，使用 Chrome 或 Edge 瀏覽器開啟。
4. 第一次開啟會有「新手導引」，建議花 30 秒跟著做一次就懂了。

### 第二步：載入照片

點擊右上角的 **「📂 選擇資料夾」**，選取您存放照片的目錄。

> **⚠️ iPhone 用戶注意**：
> 如果您的照片是 `.HEIC` 格式，Windows 瀏覽器無法預覽（會顯示全黑）。建議您在傳輸照片時，先轉存為 `.JPG` 格式再使用本工具。

### 第三步：分類與檢視

介面設計支援「左手鍵盤、右手滑鼠」，讓您極速過圖：

* **選擇主分類**：按鍵盤 `1` ~ `4` (可自訂)。
* **選擇子標籤**：按鍵盤 `A`, `S`, `D` (例如：長官、大合照)。
* **刪除廢片**：按鍵盤 `Delete`。照片會被標記並加上遮罩。
* **換下一張**：按鍵盤 `Enter` 或 `→`。
* **視圖切換**：點擊上方按鈕切換「完整 (Fit)」或「小圖 (Focus)」模式。
* **轉向**：按 `R` 可旋轉檢視（注意：這只是讓您方便預覽，**不會**修改原始檔案的方向）。
* **智慧手動命名**：
* 如果不勾選「自動流水號」，且沒有選擇任何分類，程式會**自動帶入原始檔名**到輸入框，方便您直接在原檔名上做修改。


* **重複檔名偵測 (New)**：
* 若您手動輸入的檔名與列表中其他檔案重複（例如流水號衝突），系統會即時顯示 **⚠️ 紅色警告**。


* **歷史回溯與跳轉 (New)**：
* 點擊右側「紀錄」清單中的項目，可直接跳回該張照片修改。
* 修改完畢後，點擊右下角出現的 **「⏩ 回到最新進度」** 按鈕，即可一鍵返回原本的工作進度，無需重複按下一張。



### 第四步：輸出與執行

整理完畢後，右下角有兩個按鈕：

1. **「📄 下載檔案參照表 (.html)」**：
* 會下載一份網頁檔，打開後可看到整理後的照片與新檔名。
* **亮點功能**：點擊檔名即可複製，方便貼上到 Word/PPT 報告中。
* *注意：請先執行下方的 BAT 檔歸檔照片後，參照表的圖片才顯示得出來。*


2. **「⚡ 下載整理執行檔 (.bat)」**：
* 下載後，請將 `開始整理.bat` **放在照片資料夾內**。
* 點兩下執行，系統會自動建立資料夾、移動照片並重新命名。
* **安全機制**：v8.6 版加入了防覆蓋檢查，如果目標資料夾已經有同名檔案，程式會自動跳過，確保資料安全。

## ⚙️ 進階功能說明

* **設定分類**：點擊上方的 **「⚙️ 設定分類」** 可修改您常用的分類名稱與快捷鍵。
* **鎖定分類**：點擊 **「🔓 鎖定分類」**。適合連拍場景（如連續 10 張都是會議照），下一張會自動帶入目前的分類，省去重複按鍵的時間。
* **分批處理**：若您是分批整理，可在左側設定 **「🔢 流水號起始值」**（例如設為 101），避免檔名與上一批次衝突。
* **防呆過濾**：程式會自動過濾掉檔名中的特殊符號（如 `\ / : * ? " < > |`），避免存檔失敗。

## ⚠️ 常見問題 (FAQ)

### Q: 下載的 .bat 檔被 Windows 擋住，出現藍色視窗？

這是 Windows 對網路下載腳本的標準防護。

* **解法**：點擊「其他資訊 (More info)」→ 選擇 **「仍要執行 (Run anyway)」** 即可。

### Q: 照片整理後不見了？

照片被移動到新建立的資料夾中了（例如「活動紀錄」或「_TRASH」）。本工具僅執行「移動」與「重命名」，**不會**執行永久刪除指令。

### Q: 為什麼參照表打開來圖片是破圖 (X)？

參照表是對應「整理後」的路徑。請務必先執行 `.bat` 檔，把照片歸檔到資料夾後，參照表才能正確讀取到圖片。
