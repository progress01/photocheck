# 📷 照片歸檔小工具 (Photo Archiver) v7.2

為行政人員、活動紀錄整理設計的網頁工具。
不需要安裝任何軟體，直接用瀏覽器打開即可使用。它可以幫助您快速檢視大量照片，並透過鍵盤操作完成分類與命名，最後自動產生整理腳本。

## 📌 這個工具能做什麼？

如果您手邊有幾百張活動照片，需要做以下事情：
1.  **挑選照片**：剔除模糊、重複的廢片。
2.  **分類歸檔**：依照「活動」、「會議」、「場勘」建立不同資料夾。
3.  **統一命名**：將檔名從 `DSC_1234.jpg` 改為 `活動_長官致詞_001.jpg`。

這個工具可以讓您在一個畫面中完成上述所有決定，最後幫您自動執行。

## 🛠️ 使用方法 (SOP)

### 第一步：開啟工具
1.  下載本專案的 `index.html` 檔案。
2.  直接點兩下，使用 Chrome 或 Edge 瀏覽器開啟。
3.  第一次開啟會有「新手導引」，建議花 30 秒跟著操作一次。

### 第二步：載入照片
點擊右上角的 **「📂 選擇資料夾」**，選取您存放照片的資料夾。
> **注意**：此工具是「純網頁」運作，照片只會在您的電腦瀏覽器顯示，**絕對不會上傳**到任何網路伺服器，請放心資安問題。

### 第三步：分類與檢視
介面設計讓您能「左手按鍵盤、右手按滑鼠/Enter」快速操作：

* **選擇主分類 (資料夾)**：按鍵盤 `1`, `2`, `3`, `4`。
* **選擇子標籤 (檔名後綴)**：按鍵盤 `A`, `S`, `D` (例如：長官、大合照)。
* **換下一張**：按鍵盤 `Enter` 或點擊畫面右下角的「下一張」。
* **刪除廢片**：按鍵盤 `Delete`。照片會被標記，稍後統一移至資源回收資料夾 (`_TRASH`)。
* **特殊命名**：如果不想用流水號，可以關閉右側的「自動流水號」開關，手動輸入檔名。

### 第四步：執行整理
1.  全部照片看完後，點擊右下角的橘色按鈕 **「⚡ 下載整理執行檔 (.bat)」**。
2.  瀏覽器會下載一個名為 `開始整理.bat` 的檔案。
3.  **重要**：請將這個 `.bat` 檔案，**剪下並貼上**到您原本存放照片的資料夾內。
4.  點兩下執行它，等待黑視窗跑完，整理即完成。

## ⚙️ 進階功能說明

* **自訂分類**：點擊上方的 **「⚙️ 設定」** 按鈕，您可以修改分類名稱（例如改成：工程、研習）以及對應的快捷鍵。
* **鎖定分類**：點擊 **「🔓 鎖定分類」**。適合連拍場景，下一張照片會自動帶入目前的主分類，不用重複按鍵。
* **旋轉照片**：遇到直式照片躺著時，可按 **「⟳ 轉向」** (或鍵盤 `R`) 暫時旋轉檢視。
* **復原刪除**：如果不小心按到 Delete，只要再按一次分類鍵 (如 `1`)，即可取消刪除狀態。

## ⚠️ 常見問題

### Q: 為什麼下載的 .bat 檔無法執行？電腦跳出藍色或黃色警告？
這是 Windows 系統對於網路下載檔案的標準保護機制。請依照以下步驟解除：
1.  對著 `開始整理.bat` 按 **滑鼠右鍵**。
2.  選擇 **「內容」**。
3.  勾選最下方的 **「解除鎖定 (Unblock)」**。
4.  按下 **「確定」** 即可正常執行。

### Q: 執行後，照片不見了？
照片沒有不見，只是被移動到新的資料夾了（例如「活動」或「_TRASH」）。本工具僅執行「移動 (Move)」與「改名 (Rename)」，不會執行永久刪除指令。
