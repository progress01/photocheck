## 📖 專案簡介

在處理活動紀錄、工程場勘或會議照片時，我們常面臨以下痛點：
1.  **照片數量龐大**：動輒數百張照片需要檢視。
2.  **改名繁瑣**：需要將 `IMG_8745.jpg` 改成 `活動_長官致詞_001.jpg`。
3.  **分類耗時**：需要手動建立資料夾並移動檔案。

本工具透過 **「網頁視覺化篩選 + 批次檔 (.bat) 自動執行」** 的方式，讓你在瀏覽器中用流暢速度完成分類，最後自動生成整理腳本。

---

## ✨ 核心特色

* **🔒 資安 100% 安全**：純前端運作，照片**完全不會上傳**到任何伺服器，所有動作都在你的電腦本機完成。
* **⚡ 鍵盤流**：
    * `1~4` 數字鍵：秒選主分類（建立資料夾）。
    * `A~D` 英文鍵：秒加子標籤（檔名後綴）。
    * `Enter`：下一張。
* **📂 自動化歸檔**：自動建立對應資料夾、移動檔案、並依照 `分類_標籤_流水號` 格式重新命名。
* **🗑️ 廢片回收機制**：按 `Delete` 鍵標記刪除，系統會將廢片移至 `_TRASH` 資料夾，避免誤刪原始檔。
* **🎓 內建新手駕訓班**：首次開啟會有互動式教學，確保熟悉操作後才開始工作。

## 🕹️ 操作流程 SOP

1.  **載入照片**：點擊右上角綠色按鈕 **「📂 選擇資料夾」**。
2.  **看圖分類**：
    * 左手按鍵盤 `1` (例如：活動) 決定資料夾。
    * 左手按鍵盤 `A` (例如：長官) 決定檔名後綴。
    * 右手按 `Enter` 換下一張。
3.  **下載腳本**：全部整理完後，點擊下方 **「⚡ 下載整理執行檔 (.bat)」**。
4.  **執行歸檔**：
    * 將下載的 `.bat` 檔 **放入原本的照片資料夾內**。
    * 點兩下執行，等待黑視窗跑完即大功告成！

## ⚠️ 常見問題與故障排除

### Q: 為什麼下載的 .bat 檔案無法執行？
**A:** 這是 Windows 的安全機制。因為檔案來自網路，第一次執行時會被阻擋。
請依照以下步驟 **「解除鎖定」** (只需做一次)：
1.  對著 `.bat` 檔按 **滑鼠右鍵**。
2.  選擇 **「內容 (Properties)」**。
3.  勾選最下方的 **「解除鎖定 (Unblock)」**。
4.  按 **「確定」** 即可正常執行。

### Q: 照片會被上傳嗎？
**A:** **絕對不會**。你可以斷開網路使用本工具，它依然能正常運作。這是一個純 HTML 靜態網頁。

### Q: 我不小心按錯分類怎麼辦？
**A:** 按 `←` (左箭頭) 回上一張重新選擇即可。

### Q: 按 Delete 的照片真的被刪掉了嗎？
**A:** 沒有。為了安全起見，標記刪除的照片會被移動到一個名為 `_TRASH` 的資料夾。確認不需要後，你可以手動清空該資料夾。
