透過 **「網頁視覺化篩選 + 自動生成整理腳本 (.bat)」**，讓您在瀏覽器中快速完成決策，最後一鍵執行歸檔。

---

## ✨ V6.0 核心特色

### 資安安全 (Privacy First)
* **純單機運作**：雖然是用瀏覽器開啟，但照片**完全不會上傳**到網路。
* **無毒免安裝**：不需要請資訊室安裝軟體，不需要管理者權限。

### 鍵盤工作流 (Keyboard Driven)
* **左手選分類**：按 `1~4` 秒選資料夾，按 `A~D` 秒加檔名標籤。
* **右手換頁**：按 `Enter` 下一張。
* **廢片處理**：按 `Delete` 直接丟進回收桶。
* **完全不用滑鼠**，處理 500 張照片只需 10 分鐘。

### 雙層分類系統
* **主分類 (資料夾)**：決定照片要去哪個資料夾 (例：活動、會議)。
* **子標籤 (檔名)**：決定照片的檔名後綴 (例：長官、大合照)。

## 🕹️ 操作教學 SOP

### 步驟 1：載入照片
點擊右上角的綠色按鈕 **「📂 選擇資料夾」**。

### 步驟 2：看圖分類
* **看主分類**：這是「活動」還是「會議」？按鍵盤 `1` 或 `2`。
* **看細節**：有長官嗎？按 `A`。是大合照嗎？按 `S`。
* **下一張**：確認無誤，按 `Enter`。
* **遇到爛照片**：模糊、拍壞的，直接按 `Delete` (會被移到 `_TRASH` 資料夾)。

### 步驟 3：下載執行檔
全部看完後，點擊右下角橘色按鈕 **「⚡ 下載整理執行檔 (.bat)」**。

### 步驟 4：執行歸檔 (最重要！)
1.  將下載的 `.bat` 檔，**放入原本的照片資料夾內**。
2.  點兩下執行它。
3.  等待黑視窗跑完，照片就自動分類並改名完成了！

## ⚠️ 常見問題 (Q&A)

### Q1: 為什麼 .bat 檔無法執行？出現藍色或黃色警告？
這是 Windows 的保護機制。因為檔案來自網路，第一次執行會被阻擋。
* **解法**：對檔案按 **右鍵** ➔ **內容** ➔ 勾選下方的 **「解除鎖定 (Unblock)」** ➔ **確定**。

### Q2: 執行 .bat 會把我的照片刪掉嗎？
**絕對不會**。
* 此工具只執行 `move` (移動) 和 `rename` (改名)。
* 您按 `Delete` 的照片，只是被移動到 `_TRASH` 資料夾，隨時可以救回。

### Q3: 我可以自己修改分類名稱嗎？
**可以**。
點擊介面上的 **「⚙️ 設定」** 按鈕，您可以自由新增、刪除分類，並指定您習慣的快捷鍵。
