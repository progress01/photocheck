# 📷 照片歸檔神器 (Photo Archiver) v8.2

為行政人員、活動紀錄整理設計的純網頁工具。
**無需安裝任何軟體**，直接用瀏覽器開啟 HTML 檔案即可使用。它可以幫助您快速檢視大量照片，透過鍵盤操作完成分類與命名，最後自動產生整理腳本與檔名參照表。

-----

## 📌 這個工具能做什麼？

如果您手邊有幾百張活動照片，需要處理以下繁瑣工作：

1.  **挑選照片**：剔除模糊、閉眼、重複的廢片。
2.  **分類歸檔**：依照「活動」、「會議」、「場勘」建立不同資料夾。
3.  **統一命名**：將無意義的 `DSC_1234.jpg` 改為 `活動_長官致詞_001.jpg`。
4.  **文書報告**：需要將照片檔名複製貼上到 Word 報告中。

這個工具讓您在一個畫面中完成所有決策，並自動產出執行檔與參照表。

-----

## 🛠️ 使用方法 (SOP)

### 第一步：開啟工具

1.  下載本專案的 `index.html` 檔案。
2.  **請將此檔案放入您存放照片的資料夾中**。
3.  直接點兩下，使用 Chrome 或 Edge 瀏覽器開啟。
4.  第一次開啟會有「新手導引」，建議花 30 秒熟悉操作。

### 第二步：載入照片

點擊右上角的 **「📂 選擇資料夾」**，選取您存放照片的目錄。

> **🛡️ 資安聲明**：此工具是「純網頁 (Client-side)」運作，所有照片只會在您的本機瀏覽器顯示，**絕對不會上傳**到任何網路伺服器，請放心在內網或機敏環境使用。

### 第三步：分類與檢視

介面設計支援「左手鍵盤、右手滑鼠」，讓您極速過圖：

  * **選擇主分類**：按鍵盤 `1` \~ `4` (可自訂)。
  * **選擇子標籤**：按鍵盤 `A`, `S`, `D` (例如：長官、大合照)。
  * **刪除廢片**：按鍵盤 `Delete`。照片會被標記並加上遮罩，**畫面會停留在原地供您確認**（不會自動跳下一張，避免手誤）。
  * **換下一張**：按鍵盤 `Enter` 或 `→`。
  * **視圖切換**：點擊上方按鈕切換「完整 (Fit)」、「小圖 (Focus)」或「1:1」檢視細節。
  * **分批處理設定**：若您是分批整理，可在左側設定 **「🔢 流水號起始值」**（例如設為 301），避免檔名與上一批次重複。

### 第四步：輸出與執行

整理完畢後，右下角有兩個按鈕：

1.  **「📄 下載檔案參照表 (.html)」**：

      * 會下載一份網頁檔，打開後可看到所有照片與新檔名的對照。
      * **亮點功能**：點擊按鈕即可複製檔名，方便貼上到 Word/PPT 報告中。
      * *建議：分批作業時，請手動將下載的檔案改名 (如 `參照表_Part1.html`) 以免覆蓋。*

2.  **「⚡ 下載整理執行檔 (.bat)」**：

      * 下載後，請將 `開始整理.bat` **放在照片資料夾內**。
      * 點兩下執行，系統會自動建立資料夾、移動照片並重新命名。

## ⚙️ 進階功能說明

  * **自訂與防誤觸**：點擊上方的 **「⚙️ 設定」** 可修改分類名稱。v8.2 版特別優化了刪除按鈕設計，防止手指誤觸導致視窗關閉。
  * **鎖定分類**：點擊 **「🔓 鎖定分類」**。適合連拍場景（如連續 10 張都是會議照），下一張會自動帶入目前的分類，省去重複按鍵的時間。
  * **旋轉照片**：按 **「⟳ 轉向」** (或鍵盤 `R`) 可旋轉檢視（僅供預覽，不會修改原始檔案）。
  * **命名規則開關**：若不需要自動流水號，可關閉左側開關，切換為手動輸入檔名模式。

## ⚠️ 常見問題 (FAQ)

### Q: 下載的 .bat 檔被 Windows 擋住，出現藍色視窗？

這是 Windows 對網路下載腳本的標準防護。

  * **解法**：點擊「其他資訊 (More info)」→ 選擇 **「仍要執行 (Run anyway)」** 即可。

### Q: 照片整理後不見了？

照片被移動到新建立的資料夾中了（例如「活動紀錄」或「\_TRASH」）。本工具僅執行「移動」與「重命名」，不會執行永久刪除指令。

### Q: 分批整理時，檔名會不會蓋掉舊的？

只要您在左側的 **「🔢 流水號起始值」** 設定正確（例如第一批做完 300 張，第二批設為 301），BAT 檔會自動將新照片併入同一個資料夾，與舊照片完美銜接，不會覆蓋。
